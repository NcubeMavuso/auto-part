### Adding Products
1. Go to Admin panel
2. Fill in product details:
   - Title and description
   - Price (in USD)
   - Category (select from dropdown)
   - Stock status
3. Click "Add Product"

### Modifying Categories
1. Open `script.js`
2. Find the `categories` object
3. Add new categories: `'key': 'Display Name'`
4. Add to sidebar in `index.html`
